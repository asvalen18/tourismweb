# LocalWays Bogotá

Tourism web app showcasing experiences in Bogotá, Colombia.

## Files

- `index.html` - Main page with navigation and content sections
- `css/styles.css` - Separated styles with Bogotá color scheme (blue #1b4f72, orange #f39c12)
- `img/` - Directory for local images (currently using remote sources)

## Features

✅ Banner with background image and CTA button  
✅ Sticky navigation bar  
✅ Four main sections: Routes, Culture, Gastronomy, Events  
✅ Interactive Google Maps integration  
✅ Accessibility improvements:
  - Alt attributes on all images
  - Meta description for SEO
  - Keyboard navigation (keys 1-4 to jump sections)
  - Improved focus outlines
  - ARIA labels on nav
✅ Responsive design  
✅ Hover animations

## How to Use

1. Open `index.html` in a web browser
2. Use the navigation buttons or keyboard (1-4) to explore sections
3. Click "View on map" to see locations on interactive map
